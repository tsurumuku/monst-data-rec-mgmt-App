# モンスト データ復旧フォーム管理アプリ

モンスターストライクのデータ復旧希望フォームで使う回答を、アカウントごとに保存・管理するシンプルなWebアプリです。

## できること

- アカウントを複数作成
- 指定された質問項目（バックアップ時のMIXI ID［メールアドレス形式］を含む）に沿って回答を保存
- フォーム全項目を必須入力にし、未入力では保存不可（「最後にログインした日=不明」時は日付欄を除外）
- 指定グループ（フレンドID、各モンスター欄）は重複チェックあり
- 回答内容をJSONファイルとしてエクスポート / JSONからインポート
- ローカルストレージに自動保存（ブラウザを閉じても保持）

## 使い方（おすすめ）

1. `index.html` **だけをダウンロード**してブラウザで開く
2. 左側でアカウントを追加して選択
3. 右側フォームに回答を入力して保存
4. `JSONとしてエクスポート` でバックアップ、`JSONをインポート` で復元

> `index.html` はCSS/JavaScriptを内包した単一ファイル構成なので、
> ファイル参照トラブルが起きにくく、ローカル実行向きです。

## 備考

- データ保存先は利用ブラウザのローカルストレージです。
- サーバーには送信されません。
